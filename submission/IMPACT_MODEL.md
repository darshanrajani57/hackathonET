# Impact Model (Assumption-Driven)

## Objective
Estimate business impact from faster, AI-assisted investor workflows.

## Core Assumptions
1. Active users considered: 10,000 retail investors.
2. Average analysis time saved per user per day: 45 minutes.
3. Effective value of investor time: INR 300 per hour.
4. Trading days per month: 22.
5. Missed-opportunity recovery: INR 1,500 per user per month (conservative).
6. Platform monetization (example): INR 499 per paid user per month.
7. Paid conversion from active base: 8%.

## Time Savings Value
Per user/month time saved value:
- 45 min/day = 0.75 hr/day
- Monthly hours saved = 0.75 x 22 = 16.5 hr
- Value/user/month = 16.5 x 300 = INR 4,950

Across 10,000 users:
- INR 4,950 x 10,000 = INR 49,500,000/month equivalent productivity value

## Opportunity Recovery Value
Per user/month recovered value (assumption): INR 1,500

Across 10,000 users:
- INR 1,500 x 10,000 = INR 15,000,000/month

## Combined User-Side Value
Total estimated monthly value created:
- INR 49,500,000 + INR 15,000,000 = INR 64,500,000/month

## Platform Revenue Model (Example)
Paid users:
- 10,000 x 8% = 800

Monthly recurring revenue:
- 800 x INR 499 = INR 399,200/month

Annualized revenue run-rate:
- INR 399,200 x 12 = INR 4,790,400/year

## Cost Reduction Angle (Ops/Support)
If explainable agent workflows reduce support burden by 20% for a 5-person support/analyst team:
- Assume blended monthly cost per person = INR 60,000
- Team monthly cost = INR 300,000
- 20% savings = INR 60,000/month

## Sensitivity (Conservative / Base / Aggressive)
1. Conservative
- Time saved: 25 min/day
- Opportunity recovery: INR 800/user/month

2. Base
- Time saved: 45 min/day
- Opportunity recovery: INR 1,500/user/month

3. Aggressive
- Time saved: 60 min/day
- Opportunity recovery: INR 2,500/user/month

## Notes For Judges
1. These are directional estimates, not audited financial projections.
2. Assumptions are intentionally transparent and can be stress-tested.
3. Even under conservative assumptions, user-side value remains materially positive.
